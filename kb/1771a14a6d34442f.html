>> Распределение Паскаля
 
Возвращает случайное число, распределенное по закону распределения Паскаля

function RandomPascal(k, q: Extended): Extended;
{ распределение Паскаля }
var
  tr: Extended;
  i: Integer;
begin
  tr := 1;
  q := ln(q);
  for i := 1 to Round(k) do tr := tr * Random;
  Result := ln(tr) / q;
end;

Пример использования:

x := RandomPascal(10, 5);
